# CVRP_DATA
## Attributes
- [x] Number of Costumers
- [x] Capacity

about demand:
- [x] Average demand
- [x] Standard deviation of demands
- [x] Demand-to-Capacity Ratio (i.e., the ratio of the total demand to vehicles capacity)

about general location:
- [x] Density of the instance (i.e., the number of nodes per unit hull area)
- [x] Depot Location Ratio (x and y)
- [x] Convex hull area to total area ratio

about costumer clusters after kmeans:
- [x] Number of clusters formed after elbow method
- [x] Standard deviation of cluster size
- [x] Average cluster distance to the depot
- [x] Standard deviation of cluster distance
